# 题目

给定一个字符串 s，计算具有相同数量0和1的非空(连续)子字符串的数量，并且这些子字符串中的所有0和所有1都是组合在一起的。 重复出现的子串要计算它们出现的次数。

  • 示例 1 : 输入: "00110011" 输出: 6

解释: 有6个子串具有相同数量的连续1和0：“0011”，“01”，“1100”，“10”，“0011” 和 “01”。 请注意，一些重复出现的子串要计算它们出现的次数。

另外，“00110011”不是有效的子串，因为所有的0（和1）没有组合在一起。

  • 示例 2 : 输入: "10101" 输出: 4

解释: 有4个子串：“10”，“01”，“10”，“01”，它们具有相同数量的连续1和0。

  • 注意：

    • s.length 在1到50,000之间。
    • s 只包含“0”或“1”字符。

来源：力扣696（LeetCode）

# 分析

/**
 * @param {string} s
 * @return {number}
 */
var countBinarySubstrings = function(s) {
    let arr=s.split('');
    let arr2=[];//用于存放合法字符串的
    for(let i=0,len=arr.length-1;i<len;i++){
       //提取从i开始的所有字符
        let str=test(s.slice(i));
        if(str){
            arr2.push(str);
        }
    }
    return arr2.length;
 };
function test(s) {
   // console.log(s);
     //首先正则匹配 注意匹配0或者1是整体
  //  console.log(s.match(/(0+|1+)/g));
     let str_first=s.match(/(0+|1+)/g)[0];
     //0^1=1 1^1=0相同为0不同为1 如果只有0和1，那就相当于跟1异或是自身取反
     let str_first_inverse=(str_first[0]^1).toString().repeat(str_first.length);
     //判断是否能找到这样一个东西，与他们匹配，这就需要动态创建regexp
     //注意匹配字符串的起始位置
     let exp=new RegExp(`^(${str_first}${str_first_inverse})`,'g')
     if(exp.test(s)){
         return RegExp.$1;
     }else{
         return false;
     }
}
