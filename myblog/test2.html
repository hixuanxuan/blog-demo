# 反转字符串

# 题目

给定一个字符串，你需要反转字符串中每个单词的字符顺序，同时仍保留空格和单词的初始顺序。

示例 1:

输入: "Let's take LeetCode contest" 输出: "s'teL ekat edoCteeL tsetnoc"  注意：在字符串中，每个单词由单个空格分隔，并且字符串中不会有任何额外的空格。

来源：力扣（LeetCode）

# 思路

  • 对字符串按空格进行分割，保存数组，数组元素的先后顺序就是单词的顺序
  • 对数组进行遍历，然后每个元素进行反转。 解法：
/**
 * @param {string} s
 * @return {string}
 */
var reverseWords = function(s) {
    //let arr=s.match(/\b[a-zA-Z']+\b/g)||[];
    let arr=s.split(" ");
    arr=arr.map((item,index)=>{
       return item.split('').reverse().join('');
    })
    return arr.join(' ');
};

这里面声明了变量最后返回的也是这个变量，代码更加简洁一些。

var reverseWords = function(s) {
    return s.split(' ').map((item,index)=>{
       return item.split('').reverse().join('');
    }).join(' ');
 
};

将字符串变成数组，这里面使用了split(''),当然还可以使用正则来完成，借助split或者array的match。

var reverseWords = function(s) {
    return arr.split(/\s/g).map((item,index)=>{
       return item.split('').reverse().join('');
    }).join(' ');
 
};

使用match的如下,[\w']+，是匹配单词，正向思考，把每个单词匹配出来先，我们再对单词里面的每个进行反转。

var reverseWords = function(s) {
    return s.match(/[^\s]+/g).map((item,index)=>{
       return item.split('').reverse().join('');
    }).join(' ');
 
};

这样看似可以，但letcode的测试数据中有空字符串，这样匹配还是会报错的。。。如果没有空字符串，这个方法还OK。 涉及到的方法String.prototype.split，String.prototype.match，Array原型的：map reverse join
