# 面试题目之html基础

# html元素以及功能

  1. 什么使showDow Dom（影子中的DOM） Showdow Dom是浏览器的一种功能，能够自动添加元素，例如audio，在网页中添加进度条、音量控制等功能，这些香港元素都由浏览器自动生成。

  2. meta元素可以定义文档的哪些元数据

    答案：meta元素可以定义的元数据主要可以概括为4类，如下所列

    • 声明HTML文档内容所用的字符编码
    • 完善文档描述信息，让搜索引擎更容易解析索引，提升SEO（search engine optimization）
    • 适配移动设备，使页面在各种尺寸的设备里面显示正确
    • 指定首选样式表，执行重载和重定向

  3. iframe的用途以及缺点

    • 用途有
      1. 嵌入第三方内容，例如广告，视频和音频等等，不仅能保持代码的整洁，还能保障高兼容性，并且文档之间有隔离，不用担心相互影响
      2. 移动端 hybrid App常用的jsBridge，动态添加iframe元素，再定义src属性，最后唤起某个功能。
      3. 实现长轮询，设备监控，文字直播等。这种方式是用http来模拟请求和响应，因此每次都会带上完整的请求首部（包括cookie），增加了传输量。HTML5新增的websocket，取代iframe的长轮询。
      4. 无刷新文件上传（已经被FILE API淘汰）
    • 缺点
      1. 浏览器对同一域名的并发请求数目有限制，chrome/Firefox 6个。
      2. 阻塞父窗口的load事件。
      3. script元素放在iframe之前，会阻塞iframe中的资源请求。
      4. 制造点击劫持
