# React.Children

this.props.children一般直接渲染到jsx里面就可以了，但有时候也会需要操作Children，推荐使用这个api来操作。

this.props 对象的属性与组件的属性一一对应，但是有一个例外，就是 this.props.children 属性。它表示组件的所有子节点。

const Children = {
  map,
  forEach,
  count,
  toArray,
  only,
};

源码中Children的定义，和数组类似 map和forEach类似，map会返回新数组，forEach返回原数组。
